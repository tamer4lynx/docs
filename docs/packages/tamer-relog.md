# tamer-relog

Console log capture for Lynx. Tamer4Lynx integration with auto-injection.

## Overview

Captures console output from Lynx apps and forwards it to the terminal. Used by Tamer4Lynx dev tooling for HMR and debugging.

## Installation

```bash
npm install tamer-relog
```

Add to your app's dependencies and run `t4l link`. Uses **lynx.ext.json** when configured.
