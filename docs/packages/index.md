# Packages

Tamer4Lynx packages provide routing, UI, platform APIs, and tooling for Lynx apps.

## Core

| Package | Description |
|---------|-------------|
| [tamer-dev-client](/packages/tamer-dev-client) | Dev launcher — QR scan, discovery, URL persistence, reload |
| [tamer-router](/packages/tamer-router) | File-based routing, Stack and Tabs layouts |
| [tamer-plugin](/packages/tamer-plugin) | Rsbuild plugin for tamer config |
| [tamer-app-shell](/packages/tamer-app-shell) | AppBar, TabBar, Content components |

## UI

| Package | Description |
|---------|-------------|
| [tamer-screen](/packages/tamer-screen) | Screen, SafeArea, AvoidKeyboard |
| [tamer-insets](/packages/tamer-insets) | Safe area and keyboard insets |
| [tamer-system-ui](/packages/tamer-system-ui) | Status bar, nav bar, theme colors |
| [tamer-icons](/packages/tamer-icons) | Icon component, Material/FontAwesome |
| [tamer-text-input](/packages/tamer-text-input) | Custom `<tamer-input>` element |

## Platform

| Package | Description |
|---------|-------------|
| [tamer-transports](/packages/tamer-transports) | Fetch, WebSocket, EventSource polyfills |
| [tamer-auth](/packages/tamer-auth) | OAuth 2.0 / PKCE |
| [tamer-secure-store](/packages/tamer-secure-store) | Secure key-value storage |
| [tamer-biometric](/packages/tamer-biometric) | Biometric authentication |
| [tamer-linking](/packages/tamer-linking) | Deep linking |
| [tamer-display-browser](/packages/tamer-display-browser) | In-app browser for OAuth |
